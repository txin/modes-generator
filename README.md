modes-generator
===============
To use, install the following OCaml packages:

> core, cryptokit, dolog, ocamlfind (for compilation), ocamlgraph

Then run:

> (optional) oasis setup

> ocaml setup.ml -configure

> ocaml setup.ml -build

> ./mosynth.native -src 2 -xor 1 -prf 1
